## About The Project

Dayloom is a beautifully designed, local-first web application meant to serve as a life operating system. It features an integrated environment containing:
- **Weekly Reflections**: Capture your thoughts, feelings, and what matters each week.
- **Task Management**: Fully-featured task tracker with effort/impact scoring and tagging functionality.
- **Annual Goals**: Live tracking for long-term intentions and milestones.
- **Workout Planner**: Dynamic, AI-generated exercise routines scoped to varying gym availability.
- **Insights Dashboard**: Visualize your productivity and emotional cadence over time.

### Built With

- Vanilla HTML, CSS, and JS
- [Vite](https://vitejs.dev/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need Node.js installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mlewallen/dayloom.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
4. Open the application at `http://localhost:5173`

## Features

- **Local First**: All data is efficiently stored via `localStorage` directly in your browser.
- **Google OAuth Simulation**: Quickly test out flows using dummy accounts.
- **Tagging System**: Organize everything from chores to finances visually.
- **Markdown Notes**: Embedded Markdown editors for taking rich notes on tasks and goals.
